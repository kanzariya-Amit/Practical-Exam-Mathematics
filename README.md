# 📊 Student Performance Analysis  
### Statistical Analysis of Academic Dataset  

## 📌 Project Overview  
This project focuses on analyzing a student performance dataset using core statistical concepts and linear algebra techniques. It is designed as a mini research report combining both theoretical understanding and practical implementation using Python.  

The dataset consists of ~5000 student records, including academic scores, study hours, and pass/fail outcomes. The objective is to extract meaningful insights and apply statistical methods to real-world educational data.

---

## 🧠 Part A: Theoretical Foundation  
Part A builds a strong conceptual understanding of fundamental statistical and mathematical concepts required for data analysis.

### Topics Covered:
- Mean, Median, and Mode  
- Variance and Standard Deviation  
- Normal Distribution  
- Skewness and Kurtosis  
- Probability (Empirical vs Theoretical)  
- Independent vs Dependent Events  
- Bayes Theorem  
- Eigenvalues and Eigenvectors  

Each concept is explained in a clear and concise manner with practical examples to strengthen understanding.

---

## 💻 Part B: Practical Implementation  
Part B focuses on applying statistical techniques and mathematical operations on the dataset using Python.

### Tasks Performed:

### 🔹 1. Measures of Central Tendency & Dispersion  
- Calculated mean, median, and mode for Math scores  
- Computed range, variance, and standard deviation for Science scores  

### 🔹 2. Probability Analysis  
- Calculated probability of students passing  
- Created contingency table (Pass/Fail vs Study Hours)  
- Computed conditional probability: P(Pass | Hours Studied > 5)  

### 🔹 3. Distribution & Visualization  
- Plotted histogram with normal distribution curve  
- Calculated skewness and kurtosis  
- Generated Q-Q plot to check normality  

### 🔹 4. Linear Algebra Operations  
- Represented score data as vectors  
- Calculated dot product  
- Computed L1 and L2 norms  
- Found angle between vectors  

---

## 📊 Dataset Description  
The dataset contains the following features:

- `Student_ID` – Unique identifier  
- `Age` – Age of student  
- `Math_Score` – Marks in Mathematics  
- `Science_Score` – Marks in Science  
- `English_Score` – Marks in English  
- `Hours_Studied` – Study hours per day  
- `Pass_Fail` – Binary outcome (1 = Pass, 0 = Fail)  

---

## 📊 Key Insights  
- Students who study more than 5 hours have a significantly higher probability of passing.  
- Academic scores approximately follow a normal distribution.  
- Slight skewness is observed in some subject scores.  
- Study hours show a strong influence on academic performance.  
- Statistical measures help summarize and interpret student performance effectively.
